========================================
   TraceBack Desktop - Cross Platform
========================================

IMPORTANT:
This is a cross-platform source code package that works on Windows, macOS, and Linux!

========================================
QUICK START (3 Steps)
========================================

[Windows Users]
1. Double-click: start_windows.bat
2. Wait for automatic dependency installation
3. Application opens automatically

[macOS Users]
1. Open Terminal, navigate to this directory
2. Run: chmod +x start_mac_linux.sh
3. Run: ./start_mac_linux.sh

[Linux Users]
1. Open Terminal, navigate to this directory
2. Run: chmod +x start_mac_linux.sh
3. Run: ./start_mac_linux.sh

========================================
SYSTEM REQUIREMENTS
========================================

- Python 3.11 or higher
- At least 4GB RAM
- Stable internet connection

========================================
IF YOU DON'T HAVE PYTHON
========================================

[Windows]
Visit: https://www.python.org/downloads/windows/
Download Python 3.11+, check "Add Python to PATH" during installation

[macOS]
Visit: https://www.python.org/downloads/macos/
Or use Homebrew: brew install python@3.11

[Linux]
Ubuntu/Debian: sudo apt install python3 python3-pip
CentOS/RHEL: sudo yum install python3 python3-pip

========================================
INVITATION CODE SYSTEM
========================================

- First use requires invitation code activation
- Remembers activation status, no need to re-enter
- Code is machine-bound, cannot be used on multiple computers

========================================
FAQ
========================================

Q: "Python not found" error?
A: Install Python 3.11+ and add to system PATH

Q: Dependency installation failed?
A: Manual install: cd backend && pip install -r requirements.txt

Q: Can't run .sh file on macOS?
A: First run: chmod +x start_mac_linux.sh

Q: Port already in use?
A: Modify port in backend/desktop_run.py
