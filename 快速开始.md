# TraceBack 溯·源 - 跨平台快速开始指南

## 🚀 一键启动（推荐）

### Windows 用户
双击运行：**`启动应用.bat`**

### macOS 用户
在终端中运行：
```bash
chmod +x 启动应用.sh
./启动应用.sh
```

---

## 📋 系统要求

- **Python**: 3.11 或更高版本
- **内存**: 至少 4GB RAM
- **网络**: 稳定的互联网连接

---

## 🔧 首次使用步骤

### 1. 安装 Python

#### Windows
- 访问 https://www.python.org/downloads/windows/
- 下载并安装 Python 3.11+
- ⚠️ **重要**：安装时勾选 "Add Python to PATH"

#### macOS
- 访问 https://www.python.org/downloads/macos/
- 下载并安装 Python 3.11+
- 或使用 Homebrew：`brew install python@3.11`

### 2. 启动应用

运行对应的启动脚本，脚本会自动：
- ✓ 检测 Python 环境
- ✓ 安装所需依赖包
- ✓ 启动 TraceBack 服务

### 3. 访问应用

打开浏览器访问：**http://localhost:5001**

## ❓ 常见问题

### Q: 提示"未找到Python"
A: 请确保已安装 Python 3.11+ 并添加到系统 PATH

### Q: 依赖安装失败
A: 尝试手动安装：
```bash
cd backend
pip install -r requirements.txt
```

### Q: macOS 上无法运行 .sh 文件
A: 需要先添加执行权限：
```bash
chmod +x 启动应用.sh
```

### Q: 端口被占用
A: 修改 `backend/desktop_run.py` 中的端口号
